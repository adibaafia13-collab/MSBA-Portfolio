# MSBA-Portfolio
Projects I built in the MSBA Program

1. Emerald Transformer: Strategic Site Selection
Description: Developed a weighted demand index and interactive mapping tool to identify optimal geographic locations for facility expansion using R, GIS (sf), and tidycensus data.

Impact: Automated the allocation of utility meters to counties using a 50/50 hybrid geographic and demographic weighting model, providing data-driven growth targets for the Northeast and Pacific Northwest.

2. E-Commerce Quantity Forecasting (Deep Learning)
Description: Engineered a Deep Neural Network (DNN) in TensorFlow to predict product demand using large-scale retail datasets, featuring custom hyperparameter tuning for learning rates and dropout.

Impact: Optimized inventory replenishment strategies by capturing non-linear pricing and categorical trends, reducing potential stockouts in high-volume e-commerce environments.

3. "Game-Sense" Intelligence Engine
Description: Built an NLP pipeline that scrapes live Steam reviews and applies K-Means clustering and CLIP zero-shot classification to categorize player sentiment.

Impact: Translated raw, informal gaming language into five actionable business categories (e.g., Technical Stability, Core Gameplay) to help developers prioritize patch updates.

4. Game Industry Stock Time-Series Analysis
Description: Conducted an end-to-end time-series analysis comparing EA and Take-Two stock performance using ARIMA, Prophet, and Vector Autoregression (VAR) models in R.

Impact: Evaluated cross-firm spillovers and market momentum, proving that internal firm events are stronger leading indicators for stock movement than competitor price shifts.

5. Traffic Volume Predictive Model (Hackathon)
Description: Developed a robust regression ensemble (Boosted Trees) to predict interstate traffic volume based on weather conditions and peak/off-peak seasonal trends.

Impact: Achieved high predictive accuracy (evaluated on Kaggle) to assist Departments of Transportation in scheduling maintenance during low-traffic periods.

6. Multi-Digit Recognition (Computer Vision)
Description: Implemented a Convolutional Neural Network (CNN) based on the SVHN architecture to recognize and classify multi-digit sequences in images.

Impact: Managed complex output layers that simultaneously predict both the length of a number and the specific digit at each position, achieving high exact-match accuracy.

7. Automated AI Mood Tracker & Agent
Description: Created a Generative AI system using a 7-box architecture that analyzes personal journal entries for emotional trends and provides personalized wellness tips.

Impact: Leveraged RAG (Retrieval-Augmented Generation) and local LLMs to turn unstructured text into a structured "Mood Dashboard" for behavioral pattern visualization.

8. Retail Performance Interactive Dashboard
Description: Designed a full-stack Dash/Plotly application integrated with a PostgreSQL backend to visualize sales trends, regional KPIs, and regression coefficients.

Impact: Enabled stakeholders to perform real-time "What-If" analysis on discount impacts and shipping costs through an interactive, browser-based UI.
